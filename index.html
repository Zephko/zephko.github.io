Autonomous Sailboat Challenge

Designing a motion planning system to send an autonomous robot to compete in the Vic-Maui International Yacht Race

The Context

UBC Sailbot is an engineering design team that strives to design reliable autonomous sailboats. Since I have been on the team, UBC Sailbot has been focused on building their second large-scale, fully autonomous saiboat, named Raye. The goal was for Raye to compete in the Vic-Maui Yacht Race, a 2300 nautical mile long yacht race that runs from Victoria, BC to Maui, Hawaii. Raye was to be the first fully autonomous vessel to participate in the race. The 2020 Vic-Maui Yacht Race was cancelled due to COVID-19 so we are still uncertain when Raye will compete.
I have been working on the Motion Planning Software Team. We have been building a system that will chart a course and interface with the controller being developed by our other software subteam.

The Project

Key Skills
Python (numpy, matplotlib, unittest), Open Motion Planning Library, ROS, Linux, Git, CI
